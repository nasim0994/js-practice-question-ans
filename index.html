১. জাভাস্ক্রিপ্ট কি জিনিস এইটা কি?

১২ parseInt, parseFloat, toFixed এইগুলা কি করে?
অপারেটর শর্টহ্যান্ড

parsrInt: কোনো স্ট্রিং কে নাম্বার (পুনসংখ্যা) এ রূপান্তর করার জন্য parseInt ব্যাবহার করা
var number = "10";
console.log(typeof(number)); // This is String

number = parseInt(number);
console.log(typeof(number)); // convert Number

parseFloat: কোনো স্ট্রিং কে নাম্বার(ভগনাংশ) এ রূপান্তর করার জন্য parseFloat ব্যাবহার করা
var number = "10.5";
console.log(typeof(number)); // This is String

number = parseFloat(number);
console.log(typeof(number)); // convert Number

toFxied: ভগ্নাংশ সংখ্যার দশমিক এর পর কতগুলো সংখ্যা প্রকাশ করবে তা toFixed দ্বারা সেট করা হয়।
var number = 10.5451236;
number = number.toFixed(3);
console.log(number); // output: 10.545

১৫. array এর উপাদান গুলা এর পজিশন (index) কিভাবে কাজ করে? কত দিয়ে শুরু হয়? এবং মান কিভাবে চেইঞ্জ হয়?
অ্যারে উপাদান

এরে এর ইনডেক্স 0 দিয়ে শুরু হয়।

এরে এর মান পরিবতন :
যে উপাদান এর মান পরিবতন করতে হবে সেই উপাদান এর index/posion ধরে নতুন মান বসাতে হয়।

Ex: var number = [10, 20, 50];
number[1] = 30;
console.log(number); //20 to 30

২৫. while লুপ কিভাবে কাজ করে ?
While Loop

While লুপ হল একটি চলক লুপ। এই লুপটি একবার বিবৃতিটির সত্যতা যাচাই করে ব্লকটি পরিচালনা করবে, এরপর লুপটি যতক্ষণ পর্যন্ত বিবৃতিটির সত্য ততক্ষণ পর্যন্ত লুপটি বারবার পরিচালনা করবে।

২৬. For লুপ কিভাবে কাজ করে ?
For Loop

কোন প্রোগ্রামে এক বা একাধিক স্টেটমেন্ট একটা নির্দিষ্ট সংখ্যক বার পর্যন্ত কাজ করতে জাভাস্ক্রিপ্ট এর for লুপ ব্যবহার করা হয়। অর্থাৎ যে সকল ক্ষেত্রে আমরা পূর্ব বা আগে থেকেই জানি যে জাভাস্ক্রিপ্ট কোড গুলো ঠিক কত বার পুনরাবৃত্তি করবে অর্থাৎ execute বা run করবে, সে সকল ক্ষেত্রে জাভাস্ক্রিপ্ট এর for লুপ ব্যবহার করা হয়।

৩১.while আর for loop এর মধ্যে পার্থক্য কি ?
While loop vs For loop

সিনট্যাক্সে বা আচরণ যতই ভিন্ন হোক না কেন সব লুপের উদ্দেশ্য কিন্তু একই যেমন একই কাজ বার বার করা।
তাহলে আসল পার্থক্যটা কোথায়?
আসল পার্থক্য হল তাদের ব্যবহারে।
যদিও আমরা চাইলে for কে while লুপে অথবা while কে for লুপে কনভার্ট করতে পারি কিন্তু সব লুপ সব জায়গায় ব্যবহার করা ঠিক না।

* while লুপ execute হওয়ার আগে condition সত্য হতে হয়

* for লুপ প্রথম বার execute হওয়ার জন্য একটা কাউন্টার ভেরিয়াবল initialize করতে হয়

একটা লুপ ঠিক কতবার চলবে তা যদি আগে থেকে জানা থাকে তাহলে for লুপ ব্যবহার করতে হয়। আর জানা না থাকলে while লুপ ব্যবহার করতে হয়।
